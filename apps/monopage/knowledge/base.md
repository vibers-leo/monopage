# 모노페이지 Knowledge Base
> ZeroClaw 챗봇이 참조하는 서비스 정보 문서
> 최종 업데이트: 2026-04-12

## 서비스 개요
- 서비스명: 모노페이지 (Monopage)
- URL: https://monopage.kr
- 슬로건: 나를 위한 단 하나의 페이지
- 설명: 링크만 붙여넣으면 나만의 1페이지 웹사이트가 만들어지는 서비스
- 핵심 가치: 팔로워 없음, 좋아요 없음 — 오직 나의 비즈니스에 집중
- 운영사: Vibers (vibers.co.kr)
- 가격: 현재 무료 (추후 프리미엄 플랜 예정)

## 만드는 방법
1. monopage.kr 접속 → 무료로 시작하기 클릭
2. Step 1: 프로필 사진(선택), 사용자명, 이메일, 비밀번호 입력
3. Step 2: 링크 붙여넣기 (한 개씩 또는 여러 개 한번에 가능)
4. 페이지 만들기 버튼 클릭 → 약 1~2분이면 완성
5. 완성 페이지: monopage.kr/사용자명
- 소셜 로그인 지원: 카카오, 네이버, 구글

## 지원 링크
- SNS: Instagram, YouTube, TikTok, X(Twitter), Threads, Facebook
- 블로그: 네이버 블로그
- 비즈니스: 네이버 플레이스 (자동 감지)
- 개발자: GitHub, LinkedIn
- 기타: 모든 웹사이트 URL
- 팁: @username만 입력하면 Instagram으로 자동 인식

## 주요 기능
- 프로필 관리: 사진, 바이오, 사용자명 편집
- 링크 관리: 추가, 삭제, 드래그 정렬
- 포트폴리오: 이미지 갤러리 업로드
- 레이아웃: 섹션 순서 변경, 표시/숨김
- 분석: 일별 조회수, 링크별 클릭수 통계
- 공유: QR 코드 생성, 공유 버튼
- SNS 아이콘 자동 표시
- 반응형 디자인 (모바일/PC)

## FAQ

Q: 가입비가 있나요?
A: 아니요, 현재 완전 무료입니다.

Q: 소셜 로그인만 가능한가요?
A: 이메일+비밀번호로도 가입 가능하고, 카카오/네이버/구글 소셜 로그인도 지원해요.

Q: 사용자명 변경 가능한가요?
A: 네, 어드민 대시보드 프로필 탭에서 변경할 수 있어요.

Q: 링크를 여러 개 한번에 넣을 수 있나요?
A: 네! 여러 줄 링크를 한번에 붙여넣기하면 자동 인식돼요.

Q: 링크 순서를 바꿀 수 있나요?
A: 어드민의 링크 탭에서 드래그앤드롭으로 변경 가능해요.

Q: 내 페이지 URL은?
A: monopage.kr/내사용자명 으로 접속 가능해요.

Q: 로그인이 안 돼요
A: 이메일/비밀번호를 확인해주세요. 소셜 로그인으로 가입했다면 해당 버튼을 이용해주세요.

Q: 페이지가 안 보여요
A: 주소가 정확한지 확인해주세요. 문제가 지속되면 관리자에게 문의해주세요.

Q: 사진 업로드가 안 돼요
A: JPG, PNG 파일만 지원되며 5MB 이하 이미지를 사용해주세요.

Q: 후원하고 싶어요
A: 감사합니다! 공식 후원 채널 준비 중이에요. 관리자에게 문의해주세요.

Q: 팔로우 기능이 있나요?
A: 없습니다. 팔로워/좋아요 없이 비즈니스에만 집중하는 것이 모노페이지의 철학이에요.

Q: 기업/브랜드용으로 사용할 수 있나요?
A: 물론이요! 개인, 프리랜서, 소상공인, 브랜드 누구나 가능해요.

Q: 모노페이지를 만든 곳은?
A: Vibers(바이버스)에서 만들었어요. vibers.co.kr
